## 06.03
Создан и запушен main, составленна логика git и проекта

кратко что тебе надо сделать чтобы пушить файлы.
открываешь рабочий стол ( или папку в которой будет папка проекта) -> открываешь Powershell (пкм -> открыть в терменале) -> когда первый раз хочешь работать с файлами пишешь git init потом нажимаешь enter -> заходишь в наш репозиторий видишь кнопку code она либо синяя либо зеленая -> нажимаешь на нее  -> копируешь ссылку https -> возвращаешься в терминал -> пишешь git clone вставляешь 
ссылку если все гуд будет шота типа такого <img width="995" height="169" alt="image" src="https://github.com/user-attachments/assets/82cff885-e28f-4c44-b273-d6e24d6f097c" /> -> заходишь в скопированную папку и работаешь там ( зайти в папку cd .\Python_project-_hse_media\)
после чего запомни 3 команды 
- git add . ( добавляешь все файлы ( ака все изменения в папке) )
- git commit -m "тут пишешь что изменила кратко"
- git push (заливаешь изменения в репозиторий)
